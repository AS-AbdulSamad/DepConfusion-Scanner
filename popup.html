GitHub Integration

Navigate to any GitHub repository and click the shield icon in the bottom-right corner to launch a full dependency confusion scan.

The scanner will:

  • Discover all dependency files recursively
  • Extract packages from every manifest
  • Check each against public registries
  • Flag claimable/unclaimed packages
  • Provide POC registration links

All Loaded Resources Mode

The "All Loaded Resources" scan mode fetches every JS, CSS, HTML, JSON, and SVG file loaded by any page — including third-party CDNs, analytics scripts, and cross-origin assets. It then:

  • Parses ES imports, CommonJS require(), AMD define()
  • Extracts node_modules paths from webpack bundles
  • Follows .map source maps to discover all source dependencies
  • Detects CDN references (unpkg, jsdelivr, cdnjs, skypack, esm.sh)
  • Reads import maps, data-* attributes, inline scripts & styles
  • Scans any dependency manifests served as resources
